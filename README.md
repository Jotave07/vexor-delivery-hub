# Vexor Delivery Hub

Aplicacao React para operacao de delivery proprio: vitrine publica, checkout, acompanhamento de pedidos, painel da loja e administracao da plataforma.

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- Radix UI / shadcn-ui
- Supabase
- Vitest

## Scripts

```bash
npm install
npm run dev
npm run lint
npm run test
npm run build
```
